# Python OOP Practice

A collection of beginner-friendly Python exercises focused on Object-Oriented Programming (OOP) concepts. Each task demonstrates classes, constructors (`__init__`), instance methods, and string representation (`__str__`).

## Contents

| # | Class | Concept |
|---|-------|---------|
| 1 | Person | Basic class with attributes & `__str__` |
| 2 | Car | Storing and displaying object data |
| 3 | Circle | Methods returning calculated values (area, circumference) |
| 4 | Rectangle | Area and perimeter calculation |
| 5 | Student | Calculating average from a list of marks |
| 6 | Book | Simple data class |
| 7 | Employee | Class with attributes & `__str__` |
| 8 | Bank | Deposit/withdraw logic with balance validation |

## Technologies
- Python 3.12
- Jupyter Notebook

## How to Run
1. Clone the repository
2. Open the `.ipynb` file in Jupyter Notebook or JupyterLab
3. Run the cells
